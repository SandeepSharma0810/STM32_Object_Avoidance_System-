# STM32 Object Avoidance System

## 📌 Project Description
STM32-based object avoidance system using IR sensors and L293D motor driver, implementing PWM-based motor control and real-time obstacle detection with LCD display.

## 🚀 Features
- Obstacle detection using 3 IR sensors
- Automatic navigation logic
- PWM-based motor speed control
- LCD display for real-time status

## 🛠️ Components Used
- STM32F103C6T6 Microcontroller
- IR Sensors
- L293D Motor Driver
- DC Motors
- 16x2 LCD

## 💻 Tools & Technology
- STM32CubeIDE
- Embedded C
- Proteus Simulation

## ⚙️ Working
The system detects obstacles using IR sensors. Based on sensor input, the STM32 microcontroller processes data and controls motors using PWM signals through L293D driver to avoid obstacles automatically.
